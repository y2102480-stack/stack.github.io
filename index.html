直流电路的基本概念、定律和分析方法

公式名称 公式表达式

电阻串联 R总 = R₁ + R₂ + ... + Rₙ
电阻并联 1/R总 = 1/R₁ + 1/R₂ + ... + 1/Rₙ
基尔霍夫电流定律 (KCL) ∑I入 = ∑I出

第二单元：交流电路

正弦交流电路的基本概念和计算方法

公式名称 公式表达式
正弦交流电表达式 u = Umsin(ωt + φ)
有效值与最大值关系 U = Um / √2, I = Im / √2
角频率与频率关系 ω = 2πf = 2π / T
感抗 XL = ωL = 2πfL
容抗 XC = 1/(ωC) = 1/(2πfC)
RLC串联阻抗 Z = √[R² + (XL - XC)²]
交流电路功率 P = UIcosφ, S = UI, Q = UIsinφ
